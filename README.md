# C++ Vector Processing & JSON Serialization 🚀

Цей проєкт на мові C++ демонструє роботу з динамічними масивами (`std::vector`), алгоритмами сортування, групуванням даних та збереженням результатів у формати TXT та JSON.

## 📌 Основні можливості
Програма виконує наступні кроки:
1. **Введення даних:** Користувач задає розмір вектора та його елементи.
2. **Групування:** Вектор розбивається на групи по `n` елементів, для кожної групи обчислюється сума.
3. **Обчислення:** Розрахунок специфічних параметрів $S1, S2, S3$ (різниця сум груп).
4. **Сортування:** Виконання сортування за зростанням (ASC) та спаданням (DESC).
5. **Аналіз:** Пошук дублікатів (повторюваних елементів) у масиві.
6. **Експорт:** 
   - Генерація текстового звіту в `result.txt`.
   - Збереження структурованих даних у `result.json` за допомогою власної міні-бібліотеки JSON.

## 🛠 Технології
- **Мова:** C++17
- **Стандартна бібліотека:** `iostream`, `vector`, `algorithm`, `map`, `fstream`
- **JSON:** Кастомна реалізація простору імен `nlohmann` для легкої серіалізації.

## 📂 Структура файлів
- `main.cpp` — основна логіка програми.
- `json.hpp` — заголовочний файл з логікою для форматування JSON-об'єктів.
- `result.txt` — текстовий лог виконання.
- `result.json` — підсумкові дані у форматі JSON.
